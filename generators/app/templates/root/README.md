## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production, outputs to /dist
```
npm run build
```

### Compiles and minifies for deployment, outputs to path specified in package.json
```
npm run deploy
```

### Lints and fixes files
```
npm run lint
```
